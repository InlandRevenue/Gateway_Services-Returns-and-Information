## Summary of changes

* Description of interestZeroRated modified
* Description of deductedAmount modified
* Description of grossAmount modified
* The following error codes have been removed: 978, 977, 985, 983, 995.

Schema updates

* ReturnRWT.v1.xsd
* ReturnCommon.v2.xsd
